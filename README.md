I study how machines see.  
I study how humans have always seen — through paint, through story, through markets.  
The through-line, it turns out, is pattern recognition.

<br>

I use machine learning to answer questions that art historians have been arguing about since the 1970s.  
I build AR escape rooms, pixel-art games, and things that shouldn't work but do.  
I write. Not yet published — but soon.

When I'm not at a desk: on a horse, in armor, or both.
